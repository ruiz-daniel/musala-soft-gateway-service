# gateway-service

Code Assessment for Musala Soft

# Description

Fullstack implementation for managing gateways that control multiple peripherals

Backend: Express.js

Database: in-memory MongoDB database

Frontend: React.js

# Instalation

Server side

`cd express`

install dependencies

`npm install`

run server

`npm run dev`

Client side

`cd gateway-app`

install dependencies

`npm install`

run app

`npm start`
